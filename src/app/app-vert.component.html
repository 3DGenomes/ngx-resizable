NGX-Resizable is a slit-pane layout component for workspaces.

This has been adapted from the @11mb/angular-resizable, itself an Angular2 version of @reklino/angular-resizable for AngularJS. Similar designs in vanilla JavaScript are @RickStrahl/jquery-resizable and Split.js by @nathancahill.

At 3DGenomes.org we use NGX-REsizable as one of the layout options for our 3D genomics browser TADkit.

An Angular 7+ directive for creating resizable containers.
